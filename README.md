# API Supabase Infinity

Este proyecto es una estructura base de backend moderno usando Supabase, Node.js y funciones serverless. Desarrollado con apoyo de inteligencia artificial estratégica (Infinity MetaMind X) para acelerar el desarrollo, mejorar la seguridad y facilitar la escalabilidad.

## Tecnologías utilizadas
- Supabase (PostgreSQL, Auth, Realtime)
- Node.js + TypeScript
- API REST + funciones edge
- GitHub CI/CD (en progreso)

## Estructura del proyecto
- `/api` → funciones serverless y endpoints
- `supabase.ts` → conexión con Supabase
- `index.ts` → punto de entrada del backend
